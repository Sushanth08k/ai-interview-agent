# AI Interview Preparation Agent

An Agentic AI application that helps students prepare for technical interviews through:

- Mock Interviews
- RAG-based Question Answering
- Resume-based Questions
- Weak Topic Detection
- Personalized Revision Plans

## Tech Stack

### Backend
- FastAPI
- MongoDB
- JWT Authentication

### AI
- LangChain
- LangGraph
- Gemini 2.5 Flash

### Vector Database
- ChromaDB

### Frontend
- React
